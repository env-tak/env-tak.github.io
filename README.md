## Usage

### Installation

```bash
$ git clone git@github.com:env-tak/env-tak.github.io.git
$ cd env-tak.github.io

$ npm install
```

## Running the app

```bash
$ npm start
```

## Run tests

```bash
$ npm run test
```

## Run production file locally
```bash
$ npm run build:local
$ cd dist/
# run index.html on browser
```
